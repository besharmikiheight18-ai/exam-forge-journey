# Exam Forge Journey

A modern web application for exam preparation and learning management, built with React and Lovable.

## Features

- 📚 Interactive exam preparation tools
- 🎯 Progress tracking and analytics
- 💡 Smart learning paths
- 🔐 Secure user authentication
- 📱 Responsive design

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: PostgreSQL
- **Deployment**: Vercel/Netlify

## Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/besharmikiheight18-ai/exam-forge-journey.git
cd exam-forge-journey

# Install dependencies
npm install

# Start development server
npm start
```
